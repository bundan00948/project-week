About the project

Cleaner layout, practical data focus

Macau Traffic Monitor is built to make route planning simpler: traffic estimates, camera checks, and weather alerts are now split into clear dedicated pages.

Mission

Help residents and visitors make informed travel decisions with concise, easy-to-read operational data.

Primary sources

  • SMG (weather warning and forecast RSS feeds)
  • DSAT camera portal and traffic references
  • Route-level status summaries from this site

Why the redesign

  • Replaced a long single-page interface with true multi-page navigation.
  • Introduced a shared design system for consistent style and spacing.
  • Added a dedicated SMG weather page for alerts and daily forecast view.
